What is HTML?

HTML stands for HyperTextMarkup Language.it's a markup language that web developers use to structure and describe the content of a webpage(not a programming language)

HTML consists of elements that describe different types of content;paragraphs,links,heading,images,video,etc. Web browsers understand HML and render HTML code as websites.

in HTML, each elements is mase up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element

You can learn more at: MDN Web Docs

Why should we learn HTML?

There are countless reason for lesrning the fundamental language of the web. Here are 5 of them:

  • To be able to use the fundamental dev language
  • To hand-craft beautiful websites instead of relying on tools
  • To build web applications
  • To have fun
